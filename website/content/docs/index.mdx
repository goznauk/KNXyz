title: KNXyz
description: KNX libraries for Rust, Python, and Node.js.
---

KNXyz provides KNX datapoint codecs and KNXnet/IP client building blocks for
Rust, Python, and Node.js.

Current docs: `0.0.3`

Use it to encode and decode KNX datapoint values, discover KNXnet/IP gateways,
open tunnel connections, and read or write group values.

## Install

```sh
cargo add knxyz
```

```sh
pip install knxyz
```

```sh
npm install @knxyz/knx
```
