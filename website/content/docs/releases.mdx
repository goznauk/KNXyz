## 0.0.3

KNXyz 0.0.3 expands TypeScript, Cython, and C/C++ integration.

Highlights:

- TypeScript declarations for `@knxyz/knx`
- [raw C ABI](/docs/reference/c-abi) through the Rust `knxyz` facade for
  DPT `9.001` temperature encode/decode
- Cython `cimport` support through `knxyz.capi`
- PyCapsule C API path for Python wheel consumers
